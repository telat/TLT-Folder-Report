# TLT Folder Report

**NTFS folder permission reporting, AD auditing and permission matrix tool for Windows file servers.**

## Features

- NTFS folder permission reports (Excel/PDF)
- Active Directory user & group auditing
- AD Synchronization — detects user/group changes automatically
- Permission matrix view
- Risk analysis (high/medium/low scoring)
- Delta report — changed permissions between two scans
- Inactive folders detection
- Turkish & English language support
- Offline & online license support
